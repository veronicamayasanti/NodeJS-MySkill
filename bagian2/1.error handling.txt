>>>>> Error Handling <<<<<
# Error handling adalah proses mengidentifikasi, menanggapi dan mengelolah kesalahan atau kegagaln yang mungkin terjadi selama eksekusi program
# Tujuan error handling adalah untuk memastikan bahwa program dapat mengatasi situasi yang tidak diinginkan dengan cara yang terkendali dan tidak menyebabkan kerusakan yang lebih besar
# Dalam NodeJS kita dapat menggunakan error handling dengan memanfaatkan konsep stack trace untuk melacak jejak eksekusi program saat terjadi kesalahan

>>>>> Error Async <<<<<
# Error handling pada operasi asinkron (Async) dalam NodeJS dapat dilakukan menggunakan beberapa pendekatan, 
    seperti callback, promise, atau Async/await

# Pola callback adalah Pola umum yang digunakan dalam kode NodeJS Async
# callback membuat error handling menjadi eksplisit. dengan memberikan dua parameter pada callback (error dan hasil),
    kode menjadi jelas dalam error handling
